ChAnki Setup Guide

Get up and running in 3 easy steps

1

Install AnkiConnect

ChAnki needs AnkiConnect to communicate with your Anki application.

Installation Steps:

  1. Open Anki
  2. Click Tools → Add-ons → Get Add-ons...
  3. Enter this code: 2055492159
  4. Click OK
  5. Restart Anki

💡 Tip: Anki Connect runs automatically when Anki is open. You'll see a green status indicator in ChAnki when connected.

2

Get Your Gemini API Key

ChAnki uses Google's Gemini AI to generate contextual example sentences. It's free for personal use!

Get Your Key:

  1. Visit Google AI Studio
  2. Sign in with your Google account
  3. Click "Create API Key"
  4. Copy your new API key
  5. Paste it into ChAnki Settings (⚙️ button)

⚡ Optional: ChAnki works without a Gemini key, but example sentences will use simple templates instead of AI-generated context.

3

Configure Your Deck

Tell ChAnki which Anki deck to add cards to.

Setup:

  1. In ChAnki, click the ⚙️ Settings button
  2. Select your target deck from the dropdown (e.g., "Chinese::Vocabulary")
  3. Choose your target HSK level (1-6)
  4. Enable/disable audio generation
  5. Click Save Settings

✅ Done! You're all set. ChAnki will automatically create the "ChAnki-Master" note type if it doesn't exist.

🎯 Pro Tips

🔍

Smart Search: Type in English, Pinyin (with tones like "ni3 hao3"), or Chinese characters. ChAnki searches all formats!

🎴

Dual Cards: Each word creates 2 cards - one for recall (Hanzi → meaning) and one cloze deletion (sentence with hidden word).

🏷️

Auto-Tagged: Cards are tagged with "ChAnki", "HSK-{level}", and part of speech for easy filtering in Anki.

🔊

Audio: Enable audio in settings to get native pronunciation for words and sentences.
